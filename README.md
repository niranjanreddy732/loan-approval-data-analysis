# 📊 Loan Approval Data Analysis & Prediction

## 📌 Project Overview
This project analyzes loan approval data to identify patterns, trends, and insights using data science techniques. It includes data cleaning, visualization, and basic machine learning.

---

## 🔍 Objectives
- Understand loan approval trends
- Perform data cleaning and preprocessing
- Visualize patterns using graphs
- Build a simple prediction model

---

## 🧹 Data Cleaning
- Removed empty records
- Handled missing values using forward fill
- Converted data into numeric format

---

## 📊 Exploratory Data Analysis (EDA)
- Histogram (Data distribution)
- Bar Chart (Top values)
- Pie Chart (Proportion analysis)
- Boxplot (Outlier detection)
- Scatter Plot (Trend analysis)
- Heatmap (Correlation analysis)

---

## ⚙️ Feature Engineering
- Categorized data into:
  - Low
  - Medium
  - High

---

## 🤖 Machine Learning
- Model: Linear Regression
- Train-Test Split: 80/20
- Evaluation Metrics:
  - Mean Squared Error (MSE)
  - R² Score

---

## 🛠️ Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

---

## 📈 Key Insights
- Identified patterns in loan approval data
- Detected outliers using boxplot
- Found relationships using correlation heatmap
- Built predictive model for trend analysis

---

## 🚀 Conclusion
This project demonstrates data analysis, visualization, and basic machine learning skills using real-world data.
